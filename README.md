# ThumbCraft – Professional YouTube Thumbnail Editor & Viewer

Developed by **Hariom Vishwakarma**, ThumbCraft is a powerful, browser-based tool designed to help you create stunning YouTube thumbnails and fetch existing ones with ease.

## ✨ Key Features

### 🎨 Thumbnail Editor
- **Backgrounds:** Choose from solid colors, vibrant gradients, or upload your own images.
- **Smart Video Analysis:** Upload a video, and ThumbCraft will automatically analyze it to extract 5 high-quality frames from different timestamps for you to choose as your background.
- **Text & Badges:** Add titles, subtitles, and stylized badges with full control over typography, colors, shadows, and strokes.
- **Shapes & Stickers:** Insert rectangles, circles, triangles, arrows, and emojis to make your thumbnails pop.
- **Image Overlays:** Add your own face cutouts or overlay images with ease.
- **Undo/Redo:** Full history support so you never lose your progress.
- **High-Res Export:** Download your final creation as a high-quality PNG.

### 🔍 YouTube Thumbnail Viewer
- **Universal Fetcher:** Paste any YouTube URL to instantly get all official thumbnail sizes.
- **Video Link Support:** Paste direct links to `.mp4`, `.webm`, or `.mov` files, and the tool will generate a thumbnail for you on the fly.
- **One-Click Copy:** Easily copy thumbnail URLs for use elsewhere.

## 🛠️ How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Ommii98/youtubethumbnail.git
   cd youtubethumbnail
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the server:**
   ```bash
   node server.js
   ```

4. **Open in browser:**
   Navigate to `http://localhost:3000`
